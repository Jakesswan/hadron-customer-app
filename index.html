Formulas used
pHs = (9.3 + A + B) − (C + D), where
  A = (log₁₀(TDS) − 1) / 10
  B = −13.12 · log₁₀(T + 273.15) + 34.55
  C = log₁₀(Ca hardness as CaCO₃) − 0.4
  D = log₁₀(Alkalinity as CaCO₃)
LSI = pH − pHs  ·  RSI = 2·pHs − pH
